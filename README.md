### 📂 Projetos em Destaque

#### 🎮 Hackoon GameHub (Em Desenvolvimento)
Uma infraestrutura backend robusta e isolada para gerenciamento de salas e usuários em jogos multiplayer.
* **Decoupling de Arquitetura:** Separação total das regras de negócio e modelos (`Usuario`, `Sala`) das dependências da engine de jogos, permitindo testes puros via Console Application no Rider.
* **UDP Discovery Service:** Implementação de um serviço assíncrono de escuta e envio de pacotes UDP (`UDPService`) para descoberta automática de salas na rede local via Broadcast.
* **Event-Driven UI:** Sistema baseado em eventos puros do C# (`Action`) integrado com o gerenciamento de threads do Godot (`CallDeferred`) para atualizações de interface seguras.
* **Segurança de Dados:** Validação rigorosa e padronização de inputs de usuários (como tratamento de strings e nicks) diretamente nos construtores das entidades.
